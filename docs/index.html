Datos de TMDB / RAWG · catálogo personal de Cloud Rider

Cómo funciona

Esta app lee en vivo los archivos CSV de tu repositorio Catalogo-Cloud-Rider en GitHub. Elige una categoría, busca el título, ajusta el Audio o la Categoría si quieres, y pulsa Compartir a WhatsApp: se abrirá el selector para enviar la imagen con el texto ya armado directo a tu grupo.

¿Quieres un .apk instalable de verdad, sin tocar código? Sube esta carpeta a tu hosting (por ejemplo el mismo proyecto de Vercel) y pega esa URL en pwabuilder.com. Te genera el paquete Android en un par de clics.
